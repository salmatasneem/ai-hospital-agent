# 🏥 AI Hospital Agent

An AI-powered hospital triage assistant that classifies patient symptoms and routes them to the correct department.

## 🚀 Features
- Symptom classification (General, Emergency, Mental Health)
- Built using LangGraph
- Interactive flow-based decision system

## 🧠 Example
Input:
"I am having fever"

Output:
Directed to General ward

## 🛠 Tech Stack
- Python
- LangGraph
- LLMs
